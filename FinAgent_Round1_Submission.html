1. Executive Summary

Vision

FinAgent is an intelligent Browser-Based Action Agent that transforms natural language commands into real financial actions. Unlike traditional chatbots that provide instructions, FinAgent actively performs tasksâ€”navigating websites, filling forms, and executing transactions while maintaining human oversight for critical operations.

Core Innovation

Our approach leverages Vision-Language Models (VLMs) combined with Large Action Models (LAMs) to create an agent that "sees" and "understands" web interfaces like a human, rather than relying on brittle HTML parsing. This enables adaptability to UI changes and provides a foundation for next-generation computer-use AI.

7. Error Handling Strategy

Three-Tier Error Handling Framework

Tier 1: Automatic Recovery

Error Type Detection Auto-Recovery
Slow Page Load Timeout > 10s Retry with longer wait
Element Not Found Selector fails Re-screenshot + re-find
Pop-up Interrupt Unexpected modal Close/dismiss + retry
Session Timeout Login redirect Re-authenticate

Tier 2: User Intervention

Error Type Detection User Action
Invalid Amount Negative/zero value "Please enter valid amount"
Insufficient Balance Website error msg "Balance low, proceed?"
CAPTCHA Required CAPTCHA element "Please solve CAPTCHA"
OTP Needed OTP input detected "Enter OTP from phone"

Tier 3: Graceful Abort

Error Type Detection Abort Behavior
Unrecoverable Error 3+ retry failures Log, notify, safe exit
Security Block Account locked msg Immediate stop + alert
Critical System Fail Browser crash State save + report
