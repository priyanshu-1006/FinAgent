1. Executive Summary

FinAgent is an intelligent Browser-Based Action Agent that transforms natural language commands into real financial actions. Unlike chatbots that provide instructions, FinAgent actively performs tasksâ€”navigating websites, filling forms, and executing transactions with human oversight for critical operations.

Core Innovation: We leverage Vision-Language Models (VLMs) + Large Action Models (LAMs) to create an agent that "sees" web interfaces like a human, rather than relying on brittle HTML parsing. This enables adaptability to UI changes.

4. Conscious Pause Mechanism

Core Safety Principle

The Conscious Pause ensures no high-stakes action executes without explicit human approval. This is the most critical evaluation metric.

Trigger Keywords:

  • "Pay" / "Payment" / "Transfer"
  • "Submit" / "Confirm" / "Buy"
  • "Invest" / "Delete" / "Remove"

  • No credential storage in memory
  • Session-based auth only
  • HTTPS/WSS encrypted
  • Masked logs (****1234)

  • Mandatory Conscious Pause
  • Configurable amount limits
  • Domain whitelist
  • Velocity limits

  • Visual verification per step
  • State validation
  • Explicit confirmations
