# JioFinance Bank - React App

Modern, responsive dummy banking application built with React for testing FinAgent.

## Features

- ✅ Fully responsive design (mobile-first)
- ✅ Login/Logout functionality
- ✅ Dashboard with account balance
- ✅ Pay Bills
- ✅ Fund Transfer
- ✅ Buy Digital Gold
- ✅ Profile Management
- ✅ Transaction History

## Development

```bash
# Install dependencies
npm install

# Run dev server (port 8080)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- Vite
- CSS Modules with custom properties
- Mobile-responsive design
